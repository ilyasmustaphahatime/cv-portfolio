# Ilyas Hatime CV Portfolio

A modern one-page personal CV and portfolio website built with HTML, CSS, and JavaScript. The design uses a premium dark theme with gold accents, responsive layouts, smooth animations, and beginner-friendly structure for easy editing later.

## Project Structure

```text
cv-portfolio/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   │   └── profile-placeholder.png
│   └── cv.pdf
└── README.md
```

## Features

- Responsive one-page portfolio layout
- Sticky navbar with active section highlighting
- Mobile hamburger menu
- Hero section with profile card and download CV button
- About, skills, education timeline, projects, and contact sections
- Hover effects for skills and project cards
- Scroll reveal animations
- Back-to-top button
- Placeholder project actions with friendly UI feedback
- Contact form front-end validation with success and error messages
- Subtle animated background particles

## How to Run Locally

1. Open the `cv-portfolio` folder.
2. Double-click `index.html`.
3. The website will open in your default browser.

You can also run it with a local development server if you want live reload, but it is not required.

## Easy Content Updates

- Replace `assets/images/profile-placeholder.png` with your real profile photo.
- Replace `assets/cv.pdf` with your actual CV.
- Update the placeholder contact links in the contact section.
- Add real project URLs or GitHub links inside `index.html`.

## Deploy on GitHub Pages

1. Create a new GitHub repository and upload the `cv-portfolio` folder contents.
2. Push `index.html`, `style.css`, `script.js`, `assets/`, and `README.md` to the repository.
3. In GitHub, open `Settings` for the repository.
4. Go to `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and the `/root` folder.
7. Save the settings and wait a few moments.
8. GitHub Pages will generate a public URL for your site.
